# Product Overview

My Workspace - A web application for tools work efficiently.

## Purpose
- Create bulk worklogs across date ranges for Jira tasks
- View and manage worklog history
- Quick access to admin/recurring tasks (meetings, training, etc.)
- SSE monitoring capabilities for real-time updates

## Key Features
- Session-based Jira authentication (cookie-based, no API tokens stored client-side)
- Task picker with search and status filtering
- Date range worklog creation with weekend skip option
- Worklog history viewer grouped by day
- Predefined admin tasks for common activities
- Thai language UI

## Target Users
Development teams using Jira for time tracking who need to log work across multiple days efficiently.
